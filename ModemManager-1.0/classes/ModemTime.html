get_network_time(cancellable, callback, *user_data)Â¶
Parameters:
  • cancellable (Gio.Cancellable or None) – A Gio.Cancellable or None.
  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is satisfied or None.
  • user_data (object) – User data to pass to callback.

Asynchronously requests the current network time.

When the operation is finished, callback will be invoked in the GLib.MainContext.push_thread_default of the thread you are calling this method from. You can then call ModemManager.ModemTime.get_network_time_finish() to get the result of the operation.

See ModemManager.ModemTime.get_network_time_sync() for the synchronous, blocking version of this method.
