Property Details¶

NetworkManager.SettingGsm.props.allowed_bands¶
Name:allowed-bands
Type:int
Default Value:1
Flags:READABLE, WRITABLE, CONSTRUCT

Bitfield of allowed frequency bands. Note that not all devices allow frequency band control. Permitted values are those specified by NetworkManager.SettingGsmNetworkBand.

Deprecated since version 0.9.10: No longer used. Band setting should be done by talking to ModemManager directly.

NetworkManager.SettingGsm.props.apn¶
Name:apn
Type:str
Default Value:None
Flags:READABLE, WRITABLE

The GPRS Access Point Name specifying the APN used when establishing a data session with the GSM-based network. The APN often determines how the user will be billed for their network usage and whether the user has access to the Internet or just a provider-specific walled-garden, so it is important to use the correct APN for the user’s mobile broadband plan. The APN may only be composed of the characters a-z, 0-9, ., and - per GSM 03.60 Section 14.9.

NetworkManager.SettingGsm.props.home_only¶
Name:home-only
Type:bool
Default Value:False
Flags:READABLE, WRITABLE

When True, only connections to the home network will be allowed. Connections to roaming networks will not be made.

NetworkManager.SettingGsm.props.network_id¶
Name:network-id
Type:str
Default Value:None
Flags:READABLE, WRITABLE

The Network ID (GSM LAI format, ie MCC-MNC) to force specific network registration. If the Network ID is specified, NetworkManager will attempt to force the device to register only on the specified network. This can be used to ensure that the device does not roam when direct roaming control of the device is not otherwise possible.

NetworkManager.SettingGsm.props.network_type¶
Name:network-type
Type:int
Default Value:-1
Flags:READABLE, WRITABLE, CONSTRUCT

Network preference to force the device to only use specific network technologies. The permitted values are NetworkManager.SettingGsmNetworkType.ANY, NetworkManager.SettingGsmNetworkType.UMTS_HSPA, NetworkManager.SettingGsmNetworkType.GPRS_EDGE, NetworkManager.SettingGsmNetworkType.PREFER_UMTS_HSPA, NetworkManager.SettingGsmNetworkType.PREFER_GPRS_EDGE, NetworkManager.SettingGsmNetworkType.PREFER_4G, and NetworkManager.SettingGsmNetworkType._4G. Note that not all devices allow network preference control.

Deprecated since version 0.9.10: No longer used. Network type setting should be done by talking to ModemManager directly.

NetworkManager.SettingGsm.props.number¶
Name:number
Type:str
Default Value:None
Flags:READABLE, WRITABLE

Number to dial when establishing a PPP data session with the GSM-based mobile broadband network. Many modems do not require PPP for connections to the mobile network and thus this property should be left blank, which allows NetworkManager to select the appropriate settings automatically.

NetworkManager.SettingGsm.props.password¶

Type:str
Default Value:None
Flags:READABLE, WRITABLE

The password used to authenticate with the network, if required. Many providers do not require a password, or accept any password. But if a password is required, it is specified here.
