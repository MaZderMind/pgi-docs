  • bandwidth_percent (int) – the bandwidth percentage (0 - 100) that user_priority is allowed to use within its priority group

  • enabled (bool) – True to enable flow control for this priority, False to disable it

  • bandwidth_percent (int) – the bandwidth percentage (0 - 100) to assign to group_id to

set_priority_group_id(user_priority, group_id)¶
Parameters:
  • user_priority (int) – the User Priority (0 - 7) to set flow control for
  • group_id (int) – the group (0 - 7) to assign user_priority to, or 15 for the unrestricted group.

These values are only valid when NetworkManager.SettingDcb :priority-group-flags includes the NetworkManager.SettingDcbFlags.ENABLE flag.

NetworkManager.SettingDcb.props.app_fcoe_flags¶
Name:app-fcoe-flags
Type:int
Default Value:0
Flags:READABLE, WRITABLE

Specifies the NetworkManager.SettingDcbFlags for the DCB FCoE application. Flags may be any combination of NetworkManager.SettingDcbFlags.ENABLE, NetworkManager.SettingDcbFlags.ADVERTISE, and NetworkManager.SettingDcbFlags.WILLING.

New in version 0.9.10.

NetworkManager.SettingDcb.props.app_fcoe_mode¶
Name:app-fcoe-mode
Type:str
Default Value:'fabric'
Flags:READABLE, WRITABLE, CONSTRUCT

The FCoE controller mode; either NetworkManager.SETTING_DCB_FCOE_MODE_FABRIC (default) or NetworkManager.SETTING_DCB_FCOE_MODE_VN2VN.

New in version 0.9.10.
