get_s390_option(idx)¶
Parameters:idx (int) – index of the desired option, from 0 to NetworkManager.SettingWired.get_num_s390_options() - 1
Returns:True on success if the index was valid and an option was found, False if the index was invalid (ie, greater than the number of options currently held by the setting)
out_key:on return, the key name of the s390 specific option; this value is owned by the setting and should not be modified
out_value:on return, the value of the key of the s390 specific option; this value is owned by the setting and should not be modified
Return type:(bool, out_key: str, out_value: str)

Given an index, return the value of the s390 option at that index. indexes are *not* guaranteed to be static across modifications to options done by NetworkManager.SettingWired.add_s390_option() and NetworkManager.SettingWired.remove_s390_option(), and should not be used to refer to options except for short periods of time such as during option iteration.

Property Details¶

NetworkManager.SettingWired.props.auto_negotiate¶
Name:auto-negotiate
Type:bool
Default Value:True
Flags:READABLE, WRITABLE, CONSTRUCT

If True, allow auto-negotiation of port speed and duplex mode. If False, do not allow auto-negotiation, in which case the “speed” and “duplex” properties should be set.

NetworkManager.SettingWired.props.duplex¶
Name:duplex
Type:str
Default Value:None
Flags:READABLE, WRITABLE

If specified, request that the device only use the specified duplex mode. Either “half” or “full”.

NetworkManager.SettingWired.props.mtu¶
Name:mtu
Type:int
Default Value:0
Flags:READABLE, WRITABLE, CONSTRUCT

If non-zero, only transmit packets of the specified size or smaller, breaking larger packets up into multiple Ethernet frames.

NetworkManager.SettingWired.props.port¶
Name:port
Type:str
Default Value:None
Flags:READABLE, WRITABLE

Specific port type to use if multiple the device supports multiple attachment methods. One of “tp” (Twisted Pair), “aui” (Attachment Unit Interface), “bnc” (Thin Ethernet) or “mii” (Media Independent Interface. If the device supports only one port type, this setting is ignored.

NetworkManager.SettingWired.props.s390_nettype¶
Name:s390-nettype
Type:str
Default Value:None
Flags:READABLE, WRITABLE

s390 network device type; one of “qeth”, “lcs”, or “ctc”, representing the different types of virtual network devices available on s390 systems.

NetworkManager.SettingWired.props.speed¶
Name:speed
Type:int
Default Value:0
Flags:READABLE, WRITABLE, CONSTRUCT

If non-zero, request that the device use only the specified speed. In Mbit/s, ie 100 == 100Mbit/s.
