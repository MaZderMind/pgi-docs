Property Details¶

NetworkManager.SettingInfiniband.props.mtu¶
Name:mtu
Type:int
Default Value:0
Flags:READABLE, WRITABLE, CONSTRUCT

If non-zero, only transmit packets of the specified size or smaller, breaking larger packets up into multiple frames.

NetworkManager.SettingInfiniband.props.p_key¶
Name:p-key
Type:int
Default Value:-1
Flags:READABLE, WRITABLE, CONSTRUCT

The InfiniBand P_Key to use for this device. A value of -1 means to use the default P_Key (aka “the P_Key at index 0”). Otherwise it is a 16-bit unsigned integer, whose high bit is set if it is a “full membership” P_Key.

NetworkManager.SettingInfiniband.props.parent¶
Name:parent
Type:str
Default Value:None
Flags:READABLE, WRITABLE, CONSTRUCT

The interface name of the parent device of this device. Normally None, but if the #NMSettingInfiniband:p_key property is set, then you must specify the base device by setting either this property or NetworkManager.SettingInfiniband :mac-address.

NetworkManager.SettingInfiniband.props.transport_mode¶
Name:transport-mode
Type:str
Default Value:None
Flags:READABLE, WRITABLE, CONSTRUCT

The IP-over-InfiniBand transport mode. Either “datagram” or “connected”.
