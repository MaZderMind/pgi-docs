The NetworkManager.Setting struct contains only private data. It should only be accessed through the functions described below.

classmethod new_from_hash(setting_type, hash)Â¶
Parameters:
  • setting_type (GObject.GType) – the NetworkManager.Setting type which the hash contains properties for
  • hash ({str: GObject.Value}) – the GLib.HashTable containing a string to GObject.Value mapping of properties that apply to the setting
Returns:

a new NetworkManager.Setting object populated with the properties from the hash table, or None on failure
Return type:

NetworkManager.Setting

Creates a new NetworkManager.Setting object and populates that object with the properties contained in the hash table, using each hash key as the property to set, and each hash value as the value to set that property to. Setting properties are strongly typed, thus the GObject.Value type of the hash value must be correct. See the documentation on each NetworkManager.Setting object subclass for the correct property names and value types.

clear_secrets()Â¶

Resets and clears any secrets in the setting. Secrets should be added to the setting only when needed, and cleared immediately after use to prevent leakage of information.
