NetworkManager.SettingClearSecretsWithFlagsFn(setting, secret, flags, user_data)¶
Parameters:
  • setting (NetworkManager.Setting) – The setting for which secrets are being iterated
  • secret (str) – The secret’s name
  • flags (NetworkManager.SettingSecretFlags) – The secret’s flags, eg NetworkManager.SettingSecretFlags.AGENT_OWNED
  • user_data (object or None) – User data passed to NetworkManager.Connection.clear_secrets_with_flags()
Returns:

True to clear the secret, False to not clear the secret
Return type:

bool
