class WebKit.WebPluginMIMEType¶

A structure representing one of the MIME types associated with a plugin. A GLib.SList of these objects will be returned by #webkit_web_plugin_get_mimetypes, use #webkit_web_plugin_mime_type_list_free to free it.

New in version 1.3.8.
