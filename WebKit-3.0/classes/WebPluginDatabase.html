class WebKit.WebPluginDatabase(**kwargs)¶
Bases:GObject.Object
Abstract:No
Structure:WebKit.WebPluginDatabaseClass

This object allows you to query information about the plugins found by the engine while scanning the usual directories. You can then use the WebKit.WebPlugin objects to get more information or enable/disable individual plugins.

static plugins_list_free(list)¶
Parameters:list ([WebKit.WebPlugin]) – a GLib.SList of WebKit.WebPlugin

Frees list.

New in version 1.3.8.

get_plugin_for_mimetype(mime_type)¶
Parameters:mime_type (str) – a mime type
Returns:a WebKit.WebPlugin
Return type:WebKit.WebPlugin

Returns the WebKit.WebPlugin that is handling mimeType in the self, or None if there’s none doing so.

New in version 1.3.8.

get_plugins()¶
Returns:a GLib.SList of WebKit.WebPlugin
Return type:[WebKit.WebPlugin]

Returns all WebKit.WebPlugin available in self. The returned list must be freed with WebKit.WebPluginDatabase.plugins_list_free()

New in version 1.3.8.

refresh()¶

Refreshes self adding new plugins that are now in use and removing those that have been disabled or are otherwise no longer available.

New in version 1.3.8.
