class WebKit.WebHistoryItem(**kwargs)¶
Bases:GObject.Object
Abstract:No
Structure:WebKit.WebHistoryItemClass

A history item consists out of a title and a uri. It can be part of the WebKit.WebBackForwardList and the global history. The global history is used for coloring the links of visited sites. WebKit.WebHistoryItem‘s constructed with WebKit.WebHistoryItem.new and WebKit.WebHistoryItem.new_with_data are automatically added to the global history.

/* Inject a visited page into the global history */
webkit_web_history_item_new_with_data("http://www.gnome.org/", "GNOME: The Free Software Desktop Project");
webkit_web_history_item_new_with_data("http://www.webkit.org/", "The WebKit Open Source Project");
