Class Details¶

class WebKit.Download(**kwargs)¶
Bases:GObject.Object
Abstract:No
Structure:WebKit.DownloadClass

WebKit.Download carries information about a download request, including a WebKit.NetworkRequest object. The application may use this object to control the download process, or to simply figure out what is to be downloaded, and do it itself.

classmethod new(request)¶
Parameters:request (WebKit.NetworkRequest) – a WebKit.NetworkRequest
Returns:the new WebKit.Download
Return type:WebKit.Download

Creates a new WebKit.Download object for the given WebKit.NetworkRequest object.

New in version 1.1.2.

cancel()¶

Cancels the download. Calling this will not free the WebKit.Download object, so you still need to call GObject.Object.unref() on it, if you are the owner of a reference. Notice that cancelling the download provokes the emission of the WebKit.Download ::error signal, reporting that the download was cancelled.

New in version 1.1.2.

get_destination_uri()¶
Returns:the destination URI or None
Return type:str

Obtains the URI to which the downloaded file will be written. This must have been set by the application before calling WebKit.Download.start(), and may be None.

New in version 1.1.2.

get_elapsed_time()¶
Returns:seconds since the download was started, as a float
Return type:float

Elapsed time for the download in seconds, including any fractional part. If the download is finished, had an error or was cancelled this is the time between its start and the event.

New in version 1.1.2.

get_total_size()¶
Returns:the expected total size of the downloaded file
Return type:int

Returns the expected total size of the download. This is expected because the server may provide incorrect or missing Content-Length. Notice that this may grow over time, as it will be always the same as current_size in the cases where current size surpasses it.

New in version 1.1.2.
