Class Details¶

class WebKit.WebDataSource(**kwargs)¶
Bases:GObject.Object
Abstract:No
Structure:WebKit.WebDataSourceClass

Data source encapsulates the content of a WebKit.WebFrame. A WebKit.WebFrame has a main resource and subresources and the data source provides access to these resources. When a request gets loaded initially, it is set to a provisional state. The application can request for the request that initiated the load by asking for the provisional data source and invoking the WebKit.WebDataSource.get_initial_request method of WebKit.WebDataSource. This data source may not have enough data and some methods may return empty values. To get a “full” data source with the data and resources loaded, you need to get the non-provisional data source through WebKit.WebFrame’s WebKit.WebFrame.get_data_source method. This data source will have the data after everything was loaded. Make sure that the data source was finished loading before using any of its methods. You can do this via WebKit.WebDataSource.is_loading.

classmethod new()¶
Returns:a new WebKit.WebDataSource.
Return type:WebKit.WebDataSource

Creates a new WebKit.WebDataSource instance. The URL of the WebKit.WebDataSource will be set to “about:blank”.

New in version 1.1.14.

classmethod new_with_request(request)¶
Parameters:request (WebKit.NetworkRequest) – the WebKit.NetworkRequest to use to create this data source
Returns:a new WebKit.WebDataSource
Return type:WebKit.WebDataSource

Creates a new WebKit.WebDataSource from a WebKit.NetworkRequest. Normally, WebKit.WebFrame objects create their data sources so you will almost never want to invoke this method directly.

New in version 1.1.14.

get_data()¶
Returns:a GLib.String which contains the raw data that represents the self or None if the self hasn’t loaded any data.
Return type:GLib.String

Returns the raw data that represents the the frame’s content.The data will be incomplete until the data has finished loading. Returns None if the web frame hasn’t loaded any data. Use WebKit.WebDataSource.is_loading to test if data source is in the process of loading.

New in version 1.1.14.

get_encoding()¶
Returns:the encoding name of the WebKit.WebView or of the response.
Return type:str

Returns the text encoding name as set in the WebKit.WebView, or if not, the text encoding of the response.

New in version 1.1.14.

get_initial_request()¶
Returns:the original WebKit.NetworkRequest
Return type:WebKit.NetworkRequest

Returns a reference to the original request that was used to load the web content. The WebKit.NetworkRequest returned by this method is the request prior to the “committed” load state. See WebKit.WebDataSource.get_request for getting the “committed” request.

New in version 1.1.14.

get_main_resource()¶
Returns:a new WebKit.WebResource representing the main resource of the self.
Return type:WebKit.WebResource

Returns the main resource of the self

New in version 1.1.14.

get_request()¶
Returns:the WebKit.NetworkRequest that created the self or None if the self is not attached to the frame or the frame hasn’t been loaded.
Return type:WebKit.NetworkRequest

Returns a WebKit.NetworkRequest that was used to create this WebKit.WebDataSource. The WebKit.NetworkRequest returned by this method is the request that was “committed”, and hence, different from the request you get from the WebKit.WebDataSource.get_initial_request method.

New in version 1.1.14.

get_subresources()¶
Returns:a GLib.List of WebKit.WebResource objects; the objects are owned by WebKit, but the GLib.List must be freed.
Return type:[WebKit.WebResource]

Gives you a GLib.List of WebKit.WebResource objects that compose the WebKit.WebView to which this WebKit.WebDataSource is attached.

New in version 1.1.15.

get_unreachable_uri()¶
Returns:the unreachable URL of self or None if there is no unreachable URL.
Return type:str

Return the unreachable URI of self. The self will have an unreachable URL if it was created using WebKit.WebFrame’s webkit_web_frame_load_alternate_html_string method.

New in version 1.1.14.

get_web_frame()¶
Returns:the WebKit.WebFrame that represents the self. The WebKit.WebFrame is owned by WebKit and should not be freed or destroyed. This will return None if the self is not attached to a frame.
Return type:WebKit.WebFrame

Returns the WebKit.WebFrame that represents this data source

New in version 1.1.14.

is_loading()¶
Returns:True if the self is still loading, False otherwise
Return type:bool

Determines whether the data source is in the process of loading its content.

New in version 1.1.14.
