get_message()¶
Returns:the Soup.Message
Return type:Soup.Message

Obtains the Soup.Message that represents the given response. Notice that only the response side of the HTTP conversation is represented.

New in version 1.1.14.

get_suggested_filename()¶
Returns:the suggested filename or None if not present
Return type:str

Obtains the suggested filename for the given network response. The suggested filename is taken from the ‘Content-Disposition’ HTTP header, but this is not always present, and this method will return None in such case.

New in version 1.10.

get_uri()¶
Returns:the URI of the WebKit.NetworkResponse
Return type:str

New in version 1.1.14.

set_uri(uri)¶
Parameters:uri (str) – an URI

Sets the URI held and used by the given response. When the response has an associated Soup.Message, its URI will also be set by this call.

New in version 1.1.14.
