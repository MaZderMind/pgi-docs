class GstGL.GLMemory¶

GstGL.GLMemory is a GstGL.GLBaseBuffer subclass providing support for the mapping of GL textures.

GstGL.GLMemory is created through gst_gl_memory_alloc() or system memory can be wrapped through gst_gl_memory_wrapped().

Data is uploaded or downloaded from the GPU as is necessary.

static init()¶

Initializes the GL Memory allocator. It is safe to call this function multiple times. This must be called before any other GstGL.GLMemory operation.
