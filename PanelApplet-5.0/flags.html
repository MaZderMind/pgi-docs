class PanelApplet.AppletFlags(value)¶

Bases: GObject.GFlags

Flags to be used with PanelApplet.Applet.get_flags()/PanelApplet.Applet.set_flags(), to indicate to the panel a specific behavior requested by the PanelApplet.Applet.

FLAGS_NONE = 0¶

No flags set.

EXPAND_MAJOR = 1¶

On horizontal panels, the PanelApplet.Applet will be allocated as much width as possible. On vertical panels, it will be allocated as much height as possible.

EXPAND_MINOR = 2¶

On horizontal panels, the PanelApplet.Applet will be allocated the full height of the panel. On vertical panels, it will be allocated the full width. Most applets should set this flag in order to use the full panel size and allow the applet to be Fitt’s Law compliant.

HAS_HANDLE = 4¶

The panel will draw a handle for the PanelApplet.Applet. This handle will be usable to move the applet, and to open the context menu of the applet.
