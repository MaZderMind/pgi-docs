Subclass of Clutter.Texture that displays videos using GStreamer.

The ClutterGst.VideoTexture structure contains only private data and should not be accessed directly.

static new()Â¶
Returns:the newly created video texture actor
Return type:Clutter.Actor

Creates a video texture.

This function has to be called from Clutter’s main thread. While GStreamer will spawn threads to do its work, we want all the GL calls to happen in the same thread. Clutter-gst knows which thread it is by assuming this constructor is called from the Clutter thread.

set_idle_material(material)Â¶
Parameters:material (object) – the handle of a Cogl material

Sets a material to use to draw when no media is being played. The ClutterGst.VideoTexture holds a reference of the material.

The default idle material will paint the ClutterGst.VideoTexture in black. If %COGL_INVALID_HANDLE is given as material to this function, this default idle material will be used.

New in version 1.2.
