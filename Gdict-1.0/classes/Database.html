class Gdict.Database¶

A Gdict.Database represents a database inside a dictionary source.

The Gdict.Database structure is private and should only be accessed using the available functions.

get_full_name()¶
Returns:the full name of the database. The string is owned by the Gdict.Database object, and should never be modified or freed.
Return type:str

Gets the full name of the database, suitable for display.

New in version 1.0.

get_name()¶
Returns:the short name of the database. The string is owned by the Gdict.Database object, and should never be modified or freed.
Return type:str

Gets the short name of the database, to be used with functions like Gdict.Context.match_word() or Gdict.Context.define_word().

New in version 1.0.

ref()¶
Returns:self with its reference count increased
Return type:Gdict.Database

Increases the reference count of self by one.

New in version 1.0.

unref()¶

Decreases the reference count of self by one. If the reference count reaches zero, self is destroyed.

New in version 1.0.
