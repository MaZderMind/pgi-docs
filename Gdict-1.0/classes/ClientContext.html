Gdict.Context

Creates a new Gdict.ClientContext object for hostname. Use this object to connect and query the dictionary server using the Dictionary Protocol as defined by RFC 2229.

set_client(client)¶
Parameters:client (str or None) – the client name to use, or None

Sets client as the client name to be used when advertising ourselves when a connection the the dictionary server has been established. If client is None, the default client name will be used.

set_hostname(hostname)¶
Parameters:hostname (str or None) – the hostname of a Dictionary server, or None

Sets hostname as the hostname of the dictionary server to be used. If hostname is None, the default dictionary server will be used.

set_port(port)¶
Parameters:port (int) – port of the dictionary server to be used, or -1

Sets the port of the dictionary server to be used when connecting.

If port is -1, the default port will be used.
