class Gdict.SourceLoader(**kwargs)¶

classmethod new()¶
Returns:a new Gdict.SourceLoader object
Return type:Gdict.SourceLoader

Creates a new Gdict.SourceLoader object. This object is used to search into a list of paths for dictionary source files. See Gdict.Source for more informations about the format of dictionary source files.

New in version 1.0.

add_search_path(path)¶
Parameters:path (str) – a path to be added to the search path list

Adds path to the search paths list of self.

New in version 1.0.

get_names()¶
Returns:a newly allocated, None terminated array of strings. You should free the returned string array with GLib.strfreev()
length:return location for the number of source names, or None
Return type:([str], length: int)

Retrieves the list of dictionary source names available into the search paths of self.

New in version 1.0.

get_paths()¶
Returns:a list containing the paths. The returned list is owned by the Gdict.SourceLoader object and should never be free or modified.
Return type:[str]

Gets the list of paths used by self to search for dictionary source files.

New in version 1.0.

get_source(name)¶
Parameters:name (str) – a name of a dictionary source
Returns:a referenced Gdict.Source object.
Return type:Gdict.Source

Retrieves a dictionary source using name. You can use the returned Gdict.Source object to create the right Gdict.Context for that dictionary source.

New in version 1.0.

get_sources()¶
Returns:a list of Gdict.Source objects. The returned list is owned by the Gdict.SourceLoader object, and should never be freed or modified.
Return type:[Gdict.Source]

Retrieves the list of dictionary sources available into the search paths of self, in form of Gdict.Source objects.

New in version 1.0.
