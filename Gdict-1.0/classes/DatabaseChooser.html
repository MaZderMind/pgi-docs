static new()Â¶
Returns:the newly created Gdict.DatabaseChooser widget.
Return type:Gtk.Widget

Creates a new Gdict.DatabaseChooser widget. A Database chooser widget can be used to display the list of available databases on a dictionary source using the Gdict.Context representing it. After creation, the Gdict.Context can be set using Gdict.DatabaseChooser.set_context().
