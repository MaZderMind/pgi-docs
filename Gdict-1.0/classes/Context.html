match_word(database, strategy, word)¶
Parameters:
  • database (str or None) – a database name to search into, or None for the default database
  • strategy (str or None) – a strategy name to use for matching, or None for the default strategy
  • word (str) – the word to match
Raises:

GLib.Error
Returns:

True if the query was successfully started.
Return type:

bool

Query self for a list of word matching word inside database, using strategy as a matching strategy. Each time a matching word is found, the “match-found” signal is fired.

New in version 1.0.

Gdict.Context.signals.database_lookup_end(context)¶
Signal Name:database-lookup-end
Flags:RUN_LAST
Parameters:context (Gdict.Context) – The object which received the signal

This signal is emitted when a database look up operation that has been issued using a Gdict.Context has been completed. Since every operation using a context is asynchronous, you can use this signal to know if the request has been completed or not.

New in version 1.0.
