class Gdict.Definition¶

A Gdict.Definition represents a single definition for the searched word.

The Gdict.Definition structure is private and should only be accessed using the available functions.

get_database()¶
Returns:the full name of a database. The returned string is owned by the Gdict.Definition object and should not be modified or freed.
Return type:str

Retrieves the full name of the dictionary database where the definition is stored.

New in version 1.0.

get_text()¶
Returns:the text of the definition. The returned string is owned by the Gdict.Definition object, and should not be modified or freed.
Return type:str

Retrieves the text of the definition.

New in version 1.0.

get_total()¶
Returns:the number of definitions.
Return type:int

Retrieves the total number of definitions that were found on a dictionary.

New in version 1.0.

get_word()¶
Returns:a word. The returned string is owned by the Gdict.Definition object and should not be modified or freed.
Return type:str

Retrieves the word used by the dictionary database to store the definition.

New in version 1.0.

ref()¶
Returns:the Gdict.Definition object with its reference count increased.
Return type:Gdict.Definition

Increases the reference count of self by one.

New in version 1.0.

unref()¶

Decreases the reference count of self by one. If the reference count reaches zero, the Gdict.Definition object is freed.

New in version 1.0.
