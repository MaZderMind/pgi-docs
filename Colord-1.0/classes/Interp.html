get_size()[source]¶
Returns:The original array size
Return type:int

Gets the number of items of data added with Colord.Interp.insert().

NOTE: This method has to be called after Colord.Interp.prepare() and it returns the number of items originally in the array. If you need the updated private size then just use Colord.Interp.get_x() and read the size from that.

New in version 0.1.31.

get_x()[source]¶
Returns:The X co-ordinate data
Return type:[float]

Gets the X data used the interpolation.

NOTE: this is only guaranteed to return the data inserted by Colord.Interp.insert() *BEFORE* calling Colord.Interp.prepare().

After prepating the Colord.Interp this data should be considered an implementation detail.
