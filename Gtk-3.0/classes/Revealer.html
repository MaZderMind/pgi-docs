class Gtk.Revealer(**kwargs)Â¶
Bases:Gtk.Bin
Abstract:No
Structure:Gtk.RevealerClass

The Gtk.Revealer widget is a container which animates the transition of its child from invisible to visible.

The style of transition can be controlled with Gtk.Revealer.set_transition_type().

These animations respect the Gtk.Settings :gtk-enable-animations setting.

CSS nodes

Gtk.Revealer has a single CSS node with name revealer.

The Gtk.Revealer widget was added in GTK+ 3.10.

Returns whether the child is currently revealed. See Gtk.Revealer.set_reveal_child().

This function returns True as soon as the transition is to the revealed state is started. To learn whether the child is fully revealed (ie the transition is completed), use Gtk.Revealer.get_child_revealed().
