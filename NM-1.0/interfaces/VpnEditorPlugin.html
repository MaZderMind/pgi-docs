static load_from_file(plugin_filename, check_service, check_owner, check_file, *user_data)¶
Parameters:
  • plugin_filename (str) – The path to the share library to load. Apply some common heuristics to find the library, such as appending “so” file ending. If the path is not an absolute path or no matching module can be found, lookup inside a directory defined at compile time. Due to this, check_file might be called for two different paths.
  • check_service (str) – if not-null, check that the loaded plugin advertises the given service.
  • check_owner (int) – if non-negative, check whether the file is owned by UID check_owner or by root. In this case also check that the file is not writable by anybody else.
  • check_file (NM.UtilsCheckFilePredicate) – optional callback to validate the file prior to loading the shared library.
  • user_data (object) – user data for check_file
Raises:

GLib.Error
Returns:

a new plugin instance or None on error.
Return type:

NM.VpnEditorPlugin

Load the shared libary plugin_filename and create a new NM.VpnEditorPlugin instace via the #NMVpnEditorPluginFactory function.

New in version 1.2.
