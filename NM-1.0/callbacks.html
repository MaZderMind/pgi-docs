NM.SettingClearSecretsWithFlagsFn(setting, secret, flags, user_data)¶
Parameters:
  • setting (NM.Setting) – The setting for which secrets are being iterated
  • secret (str) – The secret’s name
  • flags (NM.SettingSecretFlags) – The secret’s flags, eg NM.SettingSecretFlags.AGENT_OWNED
  • user_data (object) – User data passed to NM.Connection.clear_secrets_with_flags()
Returns:

True to clear the secret, False to not clear the secret
Return type:

bool
