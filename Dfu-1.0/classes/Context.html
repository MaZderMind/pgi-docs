set_timeout(timeout)¶
Parameters:timeout (int) – a timeout in milliseconds

Sets the wait-for-replug timeout. This is the longest we will wait for a device to re-enumerate after disconnecting. Using longer values will result in any UI not updating in a good time, but using too short values will result in devices being removed and re-added as different Dfu.Device‘s.

New in version 0.5.4.
