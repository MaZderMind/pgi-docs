set_target_size(target_size)¶
Parameters:target_size (int) – size in bytes

Sets a target size for the element. If the prepared element is smaller than this then it will be padded up to the required size.

If a padding byte other than 0x00 is required then the function Dfu.Element.set_padding_value() should be used before this function is called.

New in version 0.5.4.
