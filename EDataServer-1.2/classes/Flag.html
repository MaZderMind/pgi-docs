class EDataServer.Flag¶

Contains only private data that should be read and manipulated using the functions below.

New in version 1.12.

clear()¶

Unsets self. Subsequent calls to EDataServer.Flag.wait() or EDataServer.Flag.wait_until() will block until self is set.

New in version 1.12.

free()¶

Destroys self.

New in version 1.12.

is_set()¶
Returns:True if self is set
Return type:bool

Returns the state of self.

New in version 1.12.

set()¶

Sets self. All threads waiting on self are woken up. Threads that call EDataServer.Flag.wait() or EDataServer.Flag.wait_until() once self is set will not block at all.

New in version 1.12.

timed_wait(abs_time)¶
Parameters:abs_time (GLib.TimeVal) – a GLib.TimeVal, determining the final time
Returns:True if self is now set
Return type:bool

Blocks until self is set, or until the time specified by abs_time. If self is already set, the function returns immediately. The return value indicates the state of self after waiting.

If abs_time is None, EDataServer.Flag.timed_wait() acts like EDataServer.Flag.wait().

To easily calculate abs_time, a combination of GLib.get_current_time() and GLib.TimeVal.add() can be used.

New in version 1.12.

Deprecated since version 3.8: Use EDataServer.Flag.wait_until() instead.

wait()¶

Blocks until self is set. If self is already set, the function returns immediately.

New in version 1.12.

wait_until(end_time)¶
Parameters:end_time (int) – the monotonic time to wait until
Returns:True if self is now set
Return type:bool

Blocks until self is set, or until the time specified by end_time. If self is already set, the function returns immediately. The return value indicates the state of self after waiting.

To easily calculate end_time, a combination of GLib.get_monotonic_time() and GLib.TIME_SPAN_SECOND macro.

New in version 3.8.
