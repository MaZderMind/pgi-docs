Asks self to lookup for stored credentials for source. The out_credentials is populated with them. If the result is not None, then it should be freed with e_anmed_parameters_free() when no longer needed.

Default implementation returns False and sets Gio.IOErrorEnum.NOT_SUPPORTED error.

If an error occurs, the function sets error and returns False.
