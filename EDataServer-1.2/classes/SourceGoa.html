get_calendar_url()¶
Returns:the associated GNOME Online Account calendar URL
Return type:str

Returns the calendar URL string of the GNOME Online Account associated with the EDataServer.Source to which self belongs. Can be None or an empty string for accounts not supporting this property.

get_contacts_url()¶

set_account_id(account_id)¶
Parameters:account_id (str or None) – the associated GNOME Online Account ID, or None

Sets the identifier string of the GNOME Online Account associated with the EDataServer.Source to which self belongs.

The internal copy of account_id is automatically stripped of leading and trailing whitespace. If the resulting string is empty, None is set instead.

New in version 3.6.
