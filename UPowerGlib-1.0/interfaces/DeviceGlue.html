call_get_history(arg_type, arg_timespan, arg_resolution, cancellable, callback, *user_data)¶
Parameters:
  • arg_type (str) – Argument to pass with the method invocation.
  • arg_timespan (int) – Argument to pass with the method invocation.
  • arg_resolution (int) – Argument to pass with the method invocation.
  • cancellable (Gio.Cancellable or None) – A Gio.Cancellable or None.
  • callback (Gio.AsyncReadyCallback or None) – A Gio.AsyncReadyCallback to call when the request is satisfied or None.
  • user_data (object) – User data to pass to callback.

Asynchronously invokes the ‘GetHistory() [gdbus-method-org-freedesktop-UPower-Device.GetHistory]’ D-Bus method on self. When the operation is finished, callback will be invoked in the GLib.MainContext.push_thread_default of the thread you are calling this method from. You can then call UPowerGlib.DeviceGlue.call_get_history_finish() to get the result of the operation.

See UPowerGlib.DeviceGlue.call_get_history_sync() for the synchronous, blocking version of this method.
